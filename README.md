# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

- ネット上に配置して複数人で共有するためのリポジトリがリモートリポジトリ
- 開発者一人ひとりが使用するために自分のPC上に配置するためのリポジトリがローカルリポジトリ

## プッシュとマージの違いは何でしょうか？

- ローカルの内容をリモートにアップするのがプッシュなのに対して、マージは分岐したブランチを別のブランチに合流させること

## コミットとプッシュの違い

- 変更した履歴をローカルリポジトリに保存するのがコミット
- 変更した履歴をリモートリポジトリにアップするのがプッシュ

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

- 一目で変更内容がわかるように書く

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

- 自身のパソコン内でコードを結合する作業がローカルでマージすることであるのに対し、プルリクエストでのマージはコードのレビューを依頼し、承認を得てからリモートサーバー上で安全に結合することを指す。

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

- 以下３つの方法で変更を取り込む
    1. 先にマージされた変更内容を取り込む
    2. 後にマージしようとしている変更内容を取り込む
    3. どちらの変更内容も取り込む
